=== Immersivly-plugin for WordPress ===
Requires at least: 4.1
Tested up to: 4.1
Stable tag: 4.1

This plugin will handle all custom functionality for \"immersivly.experiencematter\" site.

== Description ==
After enabling this plugin you will have access to all the custom functionality for the immersivly.experiencematter site.

== Installation ==
Installation is simple. All you have to do is just enable the plugin, in the plugins page.
